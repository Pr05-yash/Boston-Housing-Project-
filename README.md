# Boston Housing Price Prediction

## Problem
Predict housing prices using machine learning regression models.

## Dataset
Boston Housing dataset containing features like crime rate, number of rooms, etc.

## Approach
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Regression model training
- Model evaluation

## Results
- Visualized feature distributions using normal distribution plots
- Used scatter plots to analyze relationships between variables and housing prices
- Created bar plots to compare feature impacts
