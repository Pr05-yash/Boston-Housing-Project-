# Boston Housing Price Prediction

## Problem
Predict housing prices using machine learning regression models.

## Dataset
Boston Housing dataset containing features like crime rate, number of rooms, etc.

## Approach
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Regression model training
- Model evaluation

## Results
Built regression model to predict housing prices with reasonable accuracy.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## How to Run
1. Clone repository
2. Install requirements
3. Run notebook
